OEM Key Management

Import keys in bulk (DPK, CSV, XML). Automatic distribution to technicians. Pool monitoring with low-stock email alerts.

Quality Control

Automatic hardware checks: Secure Boot, BIOS version, HackBGRT detection. Motherboard whitelist. Block activation on non-compliant builds.

Hardware Fingerprinting

SHA256 hardware fingerprint from MB serial + BIOS + disk + MAC + UUID. Detect key reuse on different hardware. Full audit trail.

One-Touch Activation

Technician plugs USB or runs CMD. Automatic: auth, hardware scan, QC check, key install, activation, report. 30 seconds per PC.

Production Tracking

Work orders, Computer Build Reports (CBR), shipping status tracking. Know exactly which key went into which machine and where it shipped.

Dynamic Task Pipeline

How It Works

1

Deploy KeyGate on your server

Docker one-liner. 4 containers: web, database, Redis, phpMyAdmin. Self-hosted — your keys never leave your network.

docker compose up -d
2

Import your OEM keys

Upload CSV, TXT, or Microsoft DPK XML files. Keys are stored encrypted, distributed one-at-a-time to technicians.

3

Technician runs the launcher on each PC

Double-click OEM_Activator.cmd. It auto-downloads the latest script, authenticates, collects hardware, runs QC, activates Windows, and reports back.

4

Track everything from the admin panel

24 pages: dashboard, keys, history, work orders, compliance, reports. Real-time visibility into your entire production line.
